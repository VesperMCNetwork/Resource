VesperCI Recipe Editor

Build secure recipes visually. Custom ingredients require genuine VesperCI items, so renamed vanilla items cannot be substituted.

Java 1.21.11–26.1.x

Create recipe

Crafting-table pattern

Each position is one crafting-table slot. Empty positions stay blank.

Ingredients (positions do not matter)

Add between one and nine ingredients. Repeating an ingredient requires repeating it in another slot.

Recipe file

Upload it to plugins/VesperCI/recipes.yml, then run /ci reload.

The editor runs entirely in your browser. It cannot access or modify your Minecraft server.
